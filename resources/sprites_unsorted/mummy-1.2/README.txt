Name:
  Mummy
Version:
  1.2
Description:
  Mummy sprites done in the style of Cabbit & based on my
  zombie sprites.
Sprite details:
  - Orientation: orthogonal
    - N/E/S/W
    - will eventually add S/W/E/N format.
  - Dimensions: 24x32 & 48x64
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
